# Mellow's Mansion — Asset Generation Task List

## Sprite Sheet Prompt Template

> A horizontal sprite sheet of 7 cartoon cat illustrations on a pure white background, evenly spaced in a single row with clear separation between each. The cat is a **[BREED]** drawn in a cute kawaii chibi style with soft pastel colors, thick outlines, and simple expressive features. Designed for a children's mobile game.
>
> From left to right, the 7 stages are:
>
> 1. **Tiny newborn kitten** — very small, eyes closed, curled into a ball, sleepy
> 2. **Small kitten** — eyes open, sitting upright, curious expression, tiny
> 3. **Playful kitten** — standing with tail up, one paw raised, excited face, slightly bigger
> 4. **Young cat** — alert sitting pose, bigger than the kittens, confident expression
> 5. **Adult cat** — full-sized, relaxed elegant pose, content smile
> 6. **Fancy cat** — same adult size, wearing a small bow or accessory, proud expression, subtle sparkle
> 7. **Majestic cat** — largest of all, wearing a tiny crown, regal seated pose, golden sparkles and a soft glow around it
>
> Each stage should be noticeably larger than the previous one. Consistent art style across all 7. Flat colors, no shading gradients, clean vector-style illustration suitable for a game UI. Each sprite fits within a square cell. No background elements, just the cats on white.

For each breed below, substitute the **[BREED]** placeholder with the breed description and generate one sprite sheet image. Save as `src/assets/breeds/{breed-name}.png`.
